# Simple Chart

Приложение создает график, который принимает 4 time-series последовательности данных.

Тогча данных имеет вид:

```ts
type DataPoint = {
  date: string;
  value: number;
}
```

## Как запустить


Установить зависимости:
```
npm install
```

Запуск:
```
npm run dev
```

## Как поменять данные?

В основной файле `App`, компонент `ChartWithData` принимает следующие пропсы:

```ts
type TChartWithData = {
  data: Record<string, DataPoint[]>;
  legends: Record<string, string>;
} 
```

`data` - сами данные графиков, пример представления:
```ts
{
  cost: [
    { date: "10.06.2026", value: 2.04 },
    { date: "11.06.2026", value: 25.85 },
    { date: "12.06.2026", value: 44.36 },
    { date: "13.06.2026", value: 55.65 },
    { date: "14.06.2026", value: 63.75 },
  ],
  cpa: ...
}
```

`legends` - подписи графиков в тултипе, пример представления:

```ts
{
  cost: "Cost",
  cpa: ...
}
```

**Важно! Для правильного отображения подписей названия ключей `legends` и `data` должны совпадать.** Если нужно поменять только данные, лучше менять только значения в `data`, не рогая ключи.
